# ToDo List

## Pourquoi ?

L'utilisateur lambda a besoin de s'organiser. Pour cela rien de tel qu'une application d'organistaion de tâches: une ToDo List !

Ce dernier pourra enregistrer des tâches à effectuer et les afficher puis les cocher lorsqu'elles auront été effectuées.  
Celles-ci apparaîtront dans la section des tâches effectuées.

Enfin, on pourra trier les tâches par date et catégories.
